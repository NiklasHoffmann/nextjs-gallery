# Galerie-Bilder

Platzieren Sie hier Ihre Galeriebilder mit den folgenden Dateinamen:

- art1.jpg
- art2.jpg
- art3.jpg
- art4.jpg
- art5.jpg

## Hinweise:

- **Format**: JPG, PNG oder WebP
- **Größe**: Empfohlen mindestens 1920x1080px für beste Qualität
- **Seitenverhältnis**: Beliebig - die Galerie passt sich automatisch an

## Beispielbilder:

Sie können kostenlose Kunstbilder von folgenden Quellen herunterladen:

1. **Unsplash**: https://unsplash.com/s/photos/art
2. **Pexels**: https://www.pexels.com/search/abstract%20art/
3. **Wikimedia Commons**: https://commons.wikimedia.org/

Oder verwenden Sie temporär Platzhalter:
- https://picsum.photos/1920/1080 (generiert zufällige Bilder)
